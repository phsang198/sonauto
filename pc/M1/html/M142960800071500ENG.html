BƯỚC 4 Kiểm tra chức năng tùy chỉnh
Kiểm tra bất kỳ một trong các điểm dưới đây "khóa: 0, Mở khóa: o" đặc cài đặt cho trả lời xác nhận đèn báo nguy bằng chức năng tùy chỉnh
  • Khóa: 1, Mở khóa: 2
  • Khóa: 1, Mở khóa: 0
  • Khóa: 0, Mở khóa: 2
  • Khóa: 2, Mở khóa: 1
  • Khóa: 2, Mở khóa: 0
  • Khóa: 0, Mở khóa: 1
  • Khóa: 0, Mở khóa: 0
